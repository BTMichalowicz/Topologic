# Topologic
A library to simulate DFAs and Probabilistic DFAs using context switching.

## Installation
Build C Library
>make

-creates libtopologic.a

Build C++ Library
>make cpp

-creates libtopologic.a

Build Python 3 Library
>make python

-creates topylogic/topylogic.so

Build Python 2 Library
>make python2

-creates topylogic/topylogic.so

Clean
>make clean
